margin 요소의 외부 여백

 margin은 요소의 테두리(border)를 기준으로 외부 여백이다. 
 양사방 존재함.
 요소의 위치와 관련됨.

 padding은 요소의 테두리(border)를 기준으로 내부 여백이다. 
 양사방 존재함.
 요소의 위치와 관련됨.

margin: n; 값이 하나로 양사방이 동일하게 적용됨

margin: 50px;
margin: 10px;
margin: 10%
margin: auto;

margin: n n; 값이 2개로 수직(상하)과 수평(좌우)이 동일하게 적용됨

margin: 100px 0;
margin: 0 100px;
margin: 50px 20%
margin: 0 auto;
가로 크기가 100%보다 작은 요소를 항상 중앙정렬하고 싶으면 로 margin: 0 auto;로 지정한다.
* 수평방향을 auto로 지정한다.

margin: n n n; 값이 3개면 '상, 수평(좌우), 하'의 순서로 적용됨

margin: 150px 50px 10px;

margin: n n n n; 값이 4개면 '상,우,하,좌(시계)'의 순서로 적용됨

margin: ;
margin: ;

margin-top: 60px;

margin-right: 3em;(48px)

margin-bottom: 30px;
