overflow: ; 영역밖의 내용 처리법

  1. overflow: visible; 영역밖의 내용 보임(기본값)
  2. overflow: hidden; 영역밖의 내용 숨김
  3. overflow: scroll; 스크롤바를 생성
  4. overflow: auto; 스크롤바 자동 생성
