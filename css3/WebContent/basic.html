CSS basic

선택자{ 속성:속성값; 속성:속성값; 속성:속성값; ... } ---- (선택자는 꾸며줄 대상, 주인공)

CSS를 적용하는 방식 3가지

  1. External CSS - 외부파일 방식
    CSS파일을 따로 만들어 독립시키고(*.css) html문서와 연결시켜 적용
    통일성 있는 디자인 구성이 가능하며 유지보수가 용이

  2. Internal CSS - 임베디드 방식(Embedded)
    CSS소스가 html문서의 head안에 존재
    해당 html문서에만 적용됨

  3. Inline CSS - 인라인 방식
    CSS소스가 html문서의 해당 태그 뒤에 존재.
    통일성 있는 디자인 구성이 어렵고, 유지보수시 불리
    (적용시 나머지 방식보다 우선순위가 가장 높음)

CSS선택자 대표 3가지

선택자의 종류가 많으며 그 중 대표적인 종류는 아래와 같다. *는 전체선택자로 말 그대로 웹문서의 모든 선택자를 불러올 수 있다.

  • tag선택자 - tag를 그대로 사용 (abc)
  • id선택자 - tag에 id를 지정하고 그 id값을 선택자로 활용 (#abc) 중복 적용 불가 / 주로 구조적인 목적에 이용
  • class선택자 - tag에 class를 지정하고 class값을 선택자로 활용 (.abc) 중복 적용 가능 / 조합가능 / 주로 디자인적인 목적에 이용
