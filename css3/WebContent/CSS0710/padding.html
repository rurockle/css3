padding - 내부 여백

  1. padding:n ; 양사방 동일(50px)
  2. padding:n n ; 상하(10px), 좌우(50px) 서로 동일
  3. padding:n n n ; 상(0), 좌우(60px) 동일, 하(20px)
  4. padding:n n n n ; 상(10px), 우(10%), 하(20px), 좌(15%)
  1. padding-top: ; 위쪽 내부여백
  2. padding-right: ; 오른쪽 내부여백
  3. padding-bottom: ; 아래쪽 내부여백
  4. padding-left: ; 왼쪽 내부여백
