키프레임(의미있는장면)들을 정의해서 디테일한 애니를 만들 수 있다.
애니메이션을 정의하고 요소에 적용한다.

애니메이션을 정의

@keyframes 애니메이션명 { -------- 애니메이션 이름을 정해줌 0%(=from) { 속성: 속성명; }---- 첫 장면에서의 상태 ... ------------- 중간 특정 장면에서의 상태 100%(=to) { 속성: 속성명; }---- 끝 장면에서의 상태 }

애니메이션 활용

  • animation-name:;(필수) 애니메이션 이름
  • animation-duration:;(필수) 전체 진행 시간
  • animation-delay:; 지연 시간
  • animation-iteration-count:; 반복 횟수
  • animation-direction:; 진행 방향)
  • animation-timing-function:; 가속 감속 등
  • animation-fill-mode:; 애니메이션이 멈춰있을 때의 장면
  • animation:; 단축형표현(속성값만 나열)

animation-direction:; 진행 방향)

  • normal 정방향(기본)
  • reverse 역방향
  • alternate 왕복
  • alternate-reverse 역방향왕복
normal
reverse
alternate
alternate-reverse

animation-timing-function:; 가속 감속 등

  • ease 기본값. 시작을 서서히 마지막을 느리게
  • ease-in 시작을 느리게
  • ease-out 마지막을 느리게
  • ease-in-out 시작을 서서히 마지막을 느리게
  • linear 등속
  • cubic-bezier(n,n,n,n) 베지에-곡선값을 이용 (cubic-bezier.com)

ease-in
ease-out
ease-in-out
