transform: ; 변형

변형함수

  1. translate() 위치 변형 : 단위(%, px, em, ...)
  2. scale() 크기 변형 : 1을 기준으로 1보다 작으면 축소, 크면 확대
  3. rotate() 회전 변형(각도) : 각도(deg) 단위
  4. skew() 기울기 변형(각도) : 각도(deg) 단위
