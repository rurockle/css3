background 배경

  • background-color:색상; 배경색상
  • background-image:url("이미지경로"); 배경이미지

  • background-repeat: ; 배경이미지 반복 지정
    1. repeat = 기본값, 양사방 반복
    2. repeat-x = 가로로만 반복
    3. repeat-y = 세로로만 반복
    4. no-repeat = 반복 안함
  • background-position: ; 배경이미지 위치 지정
    1. l/c/r t/c/b 가로 세로의 순서로 방향값을 조합해 지정
    2. npx npx / n% n% /... 가로 세로의 순서로 값을 직접 지정
      (l t or 0% 0% or 0px 0px 가 기본값. 좌측상단.)
  • background-size: ; 배경이미지 크기 지정
    1. cover = 이미지 하나로 영역을 다 덮음(이미지가 잘려나갈 수도)
    2. contain = 이미지 하나를 온전히 보이면서 최대한 영역을 다 덮음 (이미지가 반복되거나 영역이 빌 수도)
    3. npx npx / n% n% /... 가로 세로의 순서로 값을 직접 지정
  • background-attachment: ; 배경이미지 고정여부 지정
    1. scroll = 기본값. 배경이미지가 스크롤 됨
    2. fixed = 배경이미지 고정
    3. local = 특정 영역에서...
